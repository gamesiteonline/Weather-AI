# AFRICA WEATHER AI 🌍

A sophisticated, immersive weather application exclusively for African countries with AI-powered insights, glassmorphism design, and Three.js 3D visualization.

## 🌟 Features

### Core Functionality
- **Real-time Weather Data**: Accurate weather information for all African countries
- **Current Location Access**: Automatic detection and display of user's weather
- **Live Updates**: Real-time data refresh with smooth animations

### AI-Powered Insights
- **Smart Suggestions**: AI recommends events, activities, and food based on weather
- **Safety Warnings**: Real-time alerts on weather-related dangers
- **Activity Recommendations**: What to do and what to avoid
- **Predictive Analytics**: AI forecasts potential outcomes

### Design Excellence
- **Glassmorphism**: Modern frosted glass effect with blur and transparency
- **Hero State**: Dynamic temperature and weather display
- **Bento Box Grid**: Apple-inspired responsive layout
- **Dynamic Iconography**: Context-aware weather icons
- **Strict Color Theory**: Carefully curated color palette
- **Ambient Background**: Three.js particle effects
- **Micro-interactions**: Smooth animations and transitions

### Data Visualization
- **24-Hour Temperature Chart**: Visual weather trend analysis
- **Weather Details Widget**: Humidity, wind speed, pressure, visibility
- **UV Index Meter**: Sun exposure guidance
- **Air Quality Indicator**: Real-time air quality assessment
- **Sunrise/Sunset Times**: Solar event tracking
- **5-Day Forecast**: Extended weather predictions

## 🚀 Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Geolocation permissions enabled (optional)
- Internet connection for weather API

### Installation

1. **Clone or Download**
   ```bash
   cd africa-weather-ai
   ```

2. **Get API Key** (Optional - for real data)
   - Visit [OpenWeatherMap](https://openweathermap.org/api)
   - Create free account and get API key
   - Replace `demo` with your key in `script.js`

3. **Open in Browser**
   ```bash
   # Simple HTTP server
   python3 -m http.server 8000
   # Then visit: http://localhost:8000
   ```

   Or simply double-click `index.html`
