🔧 CUSTOMIZATION
================================================================================

CHANGE COLORS:
  Edit styles.css `:root`:
  --primary-color: #667eea;
  --secondary-color: #764ba2;
  --accent-color: #f093fb;

CHANGE COUNTRY:
  Edit script.js, function initializeWeatherApp():
  loadWeatherForCountry('Kenya');

ADD API KEY (FOR REAL DATA):
  1. Get key from: https://openweathermap.org/api
  2. Edit script.js line 12:
     const WEATHER_API_KEY = 'your_api_key_here';

CUSTOMIZE SUGGESTIONS:
  Edit script.js function generateAISuggestions():
  Add/modify suggestion logic
